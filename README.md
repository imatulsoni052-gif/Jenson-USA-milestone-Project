🚴 BikeStores Data Analysis Project
📌 Project Overview

This project focuses on analyzing a BikeStores database to extract meaningful business insights related to sales, customers, inventory, and store performance.

The goal is to transform raw data into actionable insights using data analysis and visualization techniques.

🎯 Objectives

Analyze sales performance across stores

Understand customer purchasing behavior

Evaluate product and brand performance

Monitor inventory levels and stock distribution

Generate insights for business decision-making

🗂️ Dataset Description

The dataset consists of multiple relational tables:

Table Name	Description
stores	Store details (location, contact info)
products	Product catalog
categories	Product categories
brands	Brand information
customers	Customer details
orders	Sales orders
order_items	Order-level details
stocks	Inventory data
staffs	Employee details
🔗 Entity Relationship Overview

One store → many staffs, orders, stocks

One customer → many orders

One order → many order items

Products linked with brands & categories
